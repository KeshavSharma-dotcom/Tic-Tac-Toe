# Tic Tac Toe - Premium Edition

A sleek, modern, and interactive Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript. This version features a "Premium Edition" aesthetic with glassmorphism effects, smooth animations, and a responsive layout.

## ✨ Features

- **Modern UI**: Beautiful glassmorphism design with backdrop filters and gradients.
- **Responsive Layout**: Optimized for both desktop and mobile devices.
- **Interactive Gameplay**: Smooth hover effects and "pop-in" animations for game pieces.
- **Game Logic**:
    - Automatic winner detection.
    - Draw detection.
    - Visual feedback for Player X and Player O.
- **Controls**: Easy "Reset" and "New Game" buttons to restart at any time.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties (CSS variables), flexbox, grid, and animations.
- **JavaScript**: ES6+ logic for game state management and DOM manipulation.
- **Google Fonts**: Uses the 'Outfit' font family for a clean, modern look.

## 🚀 How to Run

1. Clone or download this repository.
2. Open `Index.html` in any modern web browser.
3. (Optional) Use a local server (like VS Code's "Live Server" extension) for the best experience.

## 🎮 How to Play

1. The game starts with Player O.
2. Players take turns clicking on empty cells.
3. The first player to get three of their marks in a horizontal, vertical, or diagonal row wins!
4. If all nine cells are filled and no player has a row, the game is a draw.
